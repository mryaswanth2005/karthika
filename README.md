# Personal Portfolio Website — Week 1

Responsive personal portfolio website showcasing skills, projects, and contact information.

Getting started

1. Open `week1-portfolio` in VS Code.
2. Open `index.html` in the browser (Live Server recommended).

File structure

- week1-portfolio/
  - index.html
  - css/
    - variables.css
    - style.css
    - responsive.css
  - js/
    - navigation.js
  - images/
    - (add profile and project images here)

Checklist

- HTML5 semantic elements used
- External CSS files
- Mobile-first responsive styles and media queries
- Contact form with basic required fields
- Accessible attributes (alt, aria-label, aria-controls)

Next steps

- Replace placeholder images in `images/` with real screenshots
- Add more project cards and links
- Hook up form backend or integrate a service (Formspree, Netlify Forms)
